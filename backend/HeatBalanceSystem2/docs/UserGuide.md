# Руководство пользователя

## 1. Вход в систему

1) Откройте главную страницу приложения.
2) Нажмите **«Регистрация»** и создайте учётную запись.
3) Выполните **вход** по email/паролю.

## 2. Личный кабинет

После входа откройте раздел **«Личный кабинет»** (`/Cabinet`).

В личном кабинете доступно:

- создание наборов исходных данных для разных типов сушильных установок
- редактирование и удаление наборов
- запуск расчёта

## 3. Создание набора исходных данных

1) В личном кабинете нажмите **«Создать набор данных»**.
2) Выберите тип сушильной установки (конвейерная/камерная/электрическая/барабан).
3) Заполните:
   - название набора
   - комментарий (опционально)
   - исходные параметры
4) Нажмите **«Сохранить»**.

## 4. Выполнение расчёта и просмотр результатов

1) В списке наборов нажмите **«Рассчитать»**.
2) Откроется страница результатов:
   - численные итоги (подвод энергии, полезная теплота, потери, невязка, КПД)
   - таблица статей теплового баланса
   - график статей (столбчатая диаграмма)

## 5. Формирование отчёта (PDF)

На странице результатов нажмите **«Экспорт PDF»** — сформируется и скачивается отчёт в формате PDF.
